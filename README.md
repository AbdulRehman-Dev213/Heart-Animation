# ❤️ Heart Animation (HTML & CSS)

## 🌟 Project Overview
This is a simple heart animation created using only HTML and CSS.  
The project focuses on practicing CSS positioning, transforms, and animations to create visually appealing effects without using JavaScript.

---

## 🎯 Purpose of This Project
- Practice CSS animations and keyframes  
- Improve understanding of transforms and positioning  
- Explore creative UI design using pure CSS  
- Strengthen frontend fundamentals  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  

---

## ✨ Features
- ❤️ Pure CSS heart shape  
- 🎞️ Smooth animation effects  
- 🎨 Clean and minimal design  
- ⚡ No JavaScript used  
- 🧠 Lightweight and beginner-friendly  

---

## 📚 What I Learned
- CSS transform property  
- Keyframe animations  
- Positioning elements precisely  
- Creating shapes using CSS only  
- Timing and animation control  

---

## 🚀 Final Note
This project helped me understand the creative side of CSS and how simple properties can be combined to create visually engaging animations. It is part of my journey toward becoming a full-stack developer.
